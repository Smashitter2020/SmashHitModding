Documentation

Learn more about documentation for Smash Hit mods. This includes Main Game Functions (mg).

Please note that you can use main game functions with caution, such as errors including in your code, can lead to mods that contain crashes, unstable, or become unplayable, which can lead to security risks.
There are 146 Main Game Functions (mg) included for Smash Hit mods, but sometimes commands include.
mgBodyContainsPoint

mgBodyContainsPoint is a collision detection function whether a specific point in 3D space is inside a physics body that you've created.

Parameters: mgBodyContainsPoint(bodyName, x, y, z)

mgBodyDecal

myBodyDecal is a function that applies a decal texture to a physics body that adds detail to the surface.

Parameters: mgBodyDecal(bodyName, textureName)

mgBodyIsBroken

mgBodyIsBroken is a function whether or not a body is broken. If it has, then the boolean is true, if not then the boolean is false.

Parameters: myBodyIsBroken(body)

mgBodyPointToWorld

mgBodyPointToWorld is a function where it converts a point from the local coordinate system of a body to world coordinate system.

Parameters: mgBodyPointToWorld(bodyName, x, y, z)

mgBodyVecToWorld

mgBodyVecToWorld is a function where it converts to a direction vector from local space to world space.

Parameters: mgBodyVecToWorld(bodyName, x, y, z)

mgBox

mgBox is used to create visible non-textured static boxes.

Parameters: mgBox(x0, y0, z0, x1, y1, z1)

mgBoxOverlap

mgBoxOverlap is a command that is used to overlap boxes.

mgCameraDistance

mgCameraDistance is a function used to check current room distance.

mgCameraShake

mgCameraShake is used to visually shake camera in intensity.

Parameters: mgCameraShake(shake)

mgCanPass

mgCanPass is a function to determine whether the player can pass through a specific object or area.
