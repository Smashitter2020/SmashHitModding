Tutorials

Learn about tutorials when making Smash Hit mods.

Requirements

On the computer, you need APK Editor Studio, Blender, Shatter OSS, Notepad++, and Android device.

On an Android device, you need Total Commander, Smash Hit APK, Shatter Cilent, and Termux.

Warning: If it doesn't exist, such as having a computer without APK Editor Studio, it might not work properly. If you don't have one, you may need to download it from the website, make sure to have the lastest version installed.

Creating a Segment

When creating segments for Smash Hit mods, you need to have Blender installed, Shatter Cilent and OSS are required.

After you've done creating it, test the segments first, it's very important, open Shatter Cilent and then run the segment. We need to make sure that there are no errors, crashing, or unstable.

How to add custom music

To add custom music to Smash Hit mods, make sure to have the code like mgMusic.

Only use Smash Hit soundtracks if neccesary, but we need to make sure that custom music does not contain any copyrighted without permission.
