Tutorials

Learn about tutorials when making Smash Hit mods.

Requirements

On the computer, you need APK Editor Studio, Blender, Shatter OSS, Notepad++, and Android device.

On an Android device, you need Total Commander, Smash Hit APK, Shatter Cilent, and Termux.

Warning: If it doesn't exist, such as having a computer without APK Editor Studio, it might not work properly. If you don't have one, you may need to download it from the website, make sure to have the lastest version installed.

Installing a modified Smash Hit APK mod file

On an Android device

  1. Find your Android device.
  2. Download Lucky Patcher and Total Commander on your Android device for Smash Hit mods.
  3. Download the Smash Hit APK mod from trusted sources.
  4. Open Total Commander and find the Smash Hit APK mod, then click on Open as ZIP button. Make sure you have to save it, then click Yes. Modifying files can lead to blank rooms when starting on a checkpoint.
  5. Once you're done editing files, go to Lucky Patcher and then find your Smash Hit APK mod.
    1. Go to Rebuild & Install tab on Lucky Patcher.
    2. Find your Smash Hit APK mod.
    3. Once your Smash Hit APK mod is selected, click on Rebuild > Re-sign with test signature. If you try to install APK without resigning it, the installation becomes failure.
    4. Then, install the resigned Smash Hit APK mod and then install the app.

On PCs

  1. Find your computer.
  2. Extract the Smash Hit APK mod file.
  3. Download Blender, Shatter OSS, Notepad++, and APK Editor Studio on your computer.
  4. Open Blender and Shatter OSS when making segments of your own.
  5. Open APK Editor Studio to install the modified APK file.
  6. Then install the Smash Hit APK mod file on your Android device. Make sure to uninstall the original Smash Hit app first in order to backup your progress.
  7. Launch the game on an Android device.

Creating a Segment

When creating segments for Smash Hit mods, you need to have Blender installed, Shatter Cilent and OSS are required.

After you've done creating it, test the segments first, it's very important, open Shatter Cilent and then run the segment. We need to make sure that there are no errors, crashing, or unstable.

How to add custom music

To add custom music to Smash Hit mods, make sure to have the code like mgMusic.

Only use Smash Hit soundtracks if neccesary, but we need to make sure that custom music does not contain any copyrighted without permission.
