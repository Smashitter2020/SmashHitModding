Shatter Cilent

Shatter Cilent is a powerful, lets you test segments or checkpoints on an Android device from your computer.

Get started

  1. Download Shatter Cilent from Shards Community.
  2. Install the Shatter Cilent app, and then open the app.
  3. Select the IP, and set your level (e.g. test) from your computer that only exists.
  4. Make sure your device is connected to your computer.
  5. Hit Start and you'll see it works on your level!

Troubleshooting

Unable to start the game

If you still unable to start the game, because your IP is selected is invaild, set a level that does not exist, or without connection can lead to error messages.
