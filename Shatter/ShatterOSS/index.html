Shatter OSS

Shatter OSS is a powerful tool that lets you create custom segments unlike Shatter Client.

Get started

  1. Download Shatter OSS from Shards Community.
  2. Make sure you have Blender installed from the computer.
  3. Go to Edit > Preferences or Ctrl + ,.
  4. Click to the Add-ons tab.
  5. Click the arrow at the top right corner, and click Install from disk.
  6. Browse to the Shatter ZIP file that you've downloaded and open it.
