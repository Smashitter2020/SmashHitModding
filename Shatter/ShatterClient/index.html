Shatter Client is a powerful, lets you test segments or checkpoints on an Android device from your computer.

Get started

  1. Download Shatter Client from Shards Community.
  2. On your Android device, install the Shatter Client app, and then open the app.
  3. Select the IP on your computer, and set your level (e.g. test) from your computer that only exists, otherwise the client may crash. Don't select any IP on mobile devices (e.g. phones, tablets).
  4. Make sure your device is connected correctly.
  5. Hit Start and you'll see it works on your level!
  6. You can check if anything is accurate or working.

Troubleshooting

Unable to start the game

If you still unable to start the game, because your IP on your computer selected is invalid, set a level that does not exist, or without connections can lead to error messages. It might crash the client, this is because the client need to have the same network connected in order to work.

Need help on Shatter Client? Contact Support

Never submit passwords nor share your own computer IPs through Shatter Client. Anything like that can attempt to steal other information.
